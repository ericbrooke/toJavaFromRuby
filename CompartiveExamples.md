
##Method Defination 

**Ruby**
```
def foo_name(a, b)
  # stuff
end
```
**Java**
```
public static int fooName(int a, int b) {
  // stuff
}
```
1. public static : modifier.
1. int: return type or void (no return)
1. fooName: name of the method
1. a, b: formal parameters
1. int a, int b: list of parameters
